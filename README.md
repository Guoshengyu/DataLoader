# DataLoader
Semiautomatic data loader based on the similarity between input index and database index

Environment:

IDE: Intellij IDEA 13.1.4
Framework: SpringMVC 4.0.6 + Hibernate 4.2.2
Server: Tomcat 7.0.59
JDK: 1.7
Database: SQL Server 2008

Word Analyzer:
